Kubernetes

Flokkr is based on a composition based kubernetes resource manager flekszible.

With this approach you can use different configuration for different clusters.

https://github.com/flokkr/k8s repository contains the configuration set which can be combined in any way and customized. See the ./examples sub-directory for examples.

Usually you need the following:

  • install flekszible
  • Create a new directory and import required components from your flekszible.yaml
  • Generate kubernetes resources with flekszible generate
  • Use the final kubernetes resources as is
